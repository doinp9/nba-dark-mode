# Privacy Policy — NBA Dark Mode

**Last updated:** March 2026

## Summary

NBA Dark Mode does **not** collect, store, or transmit any personal data.

## Permissions

This extension requires only the `storage` permission to remember your on/off toggle preference. This data is stored locally in Chrome's sync storage and is never sent to any server.

## Data Collection

- **No analytics** — We don't track page views, clicks, or usage patterns.
- **No network requests** — The extension makes zero network requests.
- **No third-party services** — No ads, trackers, or external scripts.
- **No personal data** — We don't access browsing history, cookies, or any personal information.

## Content Script

The content script (`content.js`) runs only on `*.nba.com` domains. It modifies the visual appearance of the page (colors, backgrounds) and does not read, extract, or transmit any page content.

## Contact

For questions or concerns, please open an issue on the [GitHub repository](https://github.com/doinp9/nba-dark-mode).
